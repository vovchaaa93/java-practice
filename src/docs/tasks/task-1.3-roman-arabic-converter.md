# Задача 3: Конвертер римских чисел

## Описание
Напишите два метода для конвертации между римскими и арабскими числами.

## Требования
- **Метод**: int romanToArabic(String roman) конвертирует римское число в арабское
- **Метод**: String arabicToRoman(int arabic) конвертирует арабское число в римское (1-3999)

## Правила римских чисел:

- I=1, V=5, X=10, L=50, C=100, D=500, M=1000
- Вычитание: IV=4, IX=9, XL=40, XC=90, CD=400, CM=900
- Только валидные комбинации вычитания
- Диапазон: 1-3999

## Примеры
```java
romanToArabic("XIV") → 14
romanToArabic("MCMXC") → 1990
arabicToRoman(49) → "XLIX" 
arabicToRoman(2023) → "MMXXIII"
