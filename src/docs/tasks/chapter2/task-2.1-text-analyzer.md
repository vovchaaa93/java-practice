# Задача #2.1: АНАЛИЗАТОР ТЕКСТА

## Описание
Напишите класс для анализа текста с различной статистикой.

## Требования
- Конструктор принимает текст для анализа.
- Возвращает Map: слово -> количество вхождений.
- Возвращает самое частое слово.
- Возвращает среднюю длину слова.
- Возвращает все слова длиннее заданной.

## Правила
- Игнорировать регистр ("Привет" == "привет").
- Убирать знаки препинания (. , ! ? и т.д.).
- Слова разделяются пробелами.

## Критерии проверки
- Эффективность с большими текстами.
- Корректная обработка граничных случаев.
- Чистота кода и правильные структуры данных.

## Примеры

```java
TextAnalyzer analyzer = new TextAnalyzer("Привет мир. Привет все!");
analyzer.getWordFrequency() → {"привет": 2, "мир": 1, "все": 1}
analyzer.getMostFrequentWord() → "привет"
analyzer.getAverageWordLength() → 4.66
analyzer.getWordsLongerThan(3) → ["привет"]
