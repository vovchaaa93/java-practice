# Задача 1.2: Валидация ISBN номера

## Описание
Напишите метод для проверки валидности ISBN-10 номера.

## Требования
- **Метод**: boolean isValidISBN(String isbn)
- ISBN может содержать дефисы, но они игнорируются при проверке
- Должно быть ровно 10 цифр (последняя может быть 'X' для значения 10)
- Проверка по формуле: (10*d1 + 9*d2 + ... + 2*d9 + 1*d10) кратно 11
- Возвращает true если валиден, false если нет

## Примеры
```java
isValidISBN("0-306-40615-2") → true
isValidISBN("0306406152") → true  
isValidISBN("1-234-56789-X") → true
isValidISBN("0-306-40615-9") → false
isValidISBN("123") → false
